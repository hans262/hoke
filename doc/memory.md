## 内存
```
  1.存储方式
    计算机内存以二进制方式存储
    1kb = 1024 byte
    1byte = 8bit
    在计算机系统中，数值一律用补码来表示和存储。
    原因在于，使用补码，可以将符号位和数值位统一处理。
    计算机中的符号数有三种表示方法，即原码、反码和补码。
  2.原码
    取最高位作符号位，其余位表示数值大小。
    符号位用'0'表示正数，用'1'表示负数
    8位二进制的取值范围 = [0b1111_1111, 0b0111_1111] 即 [-127, 127]
  3.补码
    正数的补码 = 原码
    +97原码 = 0b0110_0001
    +97补码 = 0b0110_0001
    负数的补码 = 原码符号位不变 + 数值位按位取反 + 1
    -97原码 = 0b1110_0001
    -97补码 = 0b1001_1111
  4.反码
    正数的补码反码是其本身，负数的反码是符号位保持不变，其余位取反。
    +10原码 = 0b0000_1010
    +10反码 = 0b0000_1010
    负数的反码 = 原码符号位不变 + 数值位按位取反
    -10原码 = 0b1000_1010
    -10反码 = 0b1111_0101
```

## 进制表示法
```
  1.二进制 ->
  0b00000010 = 2
  0b00000011 = 3

  2.八进制 -> 0开头, 0 ~ 7
  0o10 = 8
  0o11 = 9

  3.十六进制 -> 0x开头, 0 ~ 15, A-F(0~15)
  0x10 = 16
  0x20 = 32
  0xA0 = 160
```

## 输出占位符
```
  %d int
  %f float
  %ld long
  %lld long long
  %c char
  %p 指针
  %s 字符串

  %lf double
  
  %o 八进制
  %x 十六进制
  %% 百分号

  %u unsigned int
  %lu unsigned long
```


## 类型强转
```
  (type_name) expression

  例子：
  int sum = 17, count = 5;
  double mean;
  mean = (double)sum / (double)count;
  printf("%f", mean);
```
