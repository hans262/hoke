# 数据类型

## 基本类型
其值不可以再分解其他类型，基本数据类型是自我说明的。
```
  1.数值类型
    1.整型：
    （在不同的系统上，类型占据的字节长度是不同的） sizeof(int)
    短整型 short int | short
    整型 int
      占4个字节
    长整型 long int | long
      后缀 L | l 来表示

    无符号型 unsigned
      不包含负数
      0 ~ pow(2, sizeof(int) * 8) - 1
    有符号型 signed
      声明变量时不指明signed还是unsigned，则默认为signed
      pow(2, sizeof(int) * 8)/2 * -1 ~ pow(2, sizeof(int) * 8)/2 - 1

    2.浮点型 | 实型
    单精度 float
    双精度 double
    长双精度 long double

    -十进制小数形式 必须有小数点
    0.0 25.0 5.669 0.25 
    -指数形式 加阶码 e | E
    aEn (a为十进制数，n为十进制整数)
    2.1E5 (2.1 * 10的5次幂)
    3.7E-2 (3.7 * 10的-2次幂)

    浮点数允许使用后缀：f | F
    如356f和356.是等价的

    -在内存中的存放形式：
    4个字节，32位内存空间。按指数形式存储。
    3.14159在内存中存放形式如下
    符号  小数部分  指数部分
    +     .314159  1
    小数部分占的位数越多，精度越高
    指数部分占得位越多，数值范围越大

    作业：算各个类型数值范围、有效数值范围、字节数

  2.字符类型
    char

  4.补充
    内存中的二进制存储
    一个字节 byte = 8 bit
    数值是已补码表示的
    正数的补码和原码相同
    负数的补码：将该数的绝对值的二进制，按位取反再加1
    
    例如：求-10的补码       8 bit
    10的原码：             00001010
    取反：                 11110101
    加1，得-10的补码:      11110110

  5.变量
    其值可以改变，必须先定义再使用
    类型 名称 值  0x0010ff 变量的内存地址
    int k = 3

  6.常量
    程序执行过程中，其值不发生改变
    #define 名称 值

    -十进制 -> 0 ~ 9
      范围 0 ~ 65535 | -32768 ~ 32767
    -八进制 -> 0开头，0 ~ 7
      015(十进制为13) 0101(十进制为65) 0177777(十进制为65635)
      范围 0 ~ 0177777
    -十六进制 -> 0X或0x开头，0 ~ 9 | A ~ F | a ~ f
      0X2A(十进制为42) 0XA0(十进制为160) 0XFFFF(十进制为65535)
      范围 0X0 ~ 0XFFFF | 0x0 ~ 0xFFFF

```

```
  - 数组
  - 结构体 struct
  - 共用体(联合) union
  - 枚举 enum
```
