# 数据类型

```
  * 基本数据类型
    其值不可以再分解其他类型，基本数据类型是自我说明的
  
  1.常量
    程序执行过程中，其值不发生改变

    #define 名称 值

    十进制 -> 0 ~ 9
    八进制 -> 0开头，0 ~ 7
      015(十进制为13) 0101(十进制为65) 0177777(十进制为65635)
    十六进制 -> 0X或0x开头，0 ~ 9 | A ~ F | a ~ f
      0X2A(十进制为42) 0XA0(十进制为160) 0XFFFF(十进制为65535)

    范围 -> 16位
    十进制 0 ~ 65535 | -32768 ~ +32769
    八进制 0 ~ 0177777
    十六进制 0X0 ~ 0XFFFF | 0x0 ~ 0xFFFF

    长整型 -> 超过上述的范围
    后缀：L | l 来表示


  2.变量
    其值可以改变，必须先定义再使用
    类型 名称 值  0x0010ff 变量的地址
    int k = 3

    整型：
    （这里的多少个字节，跟系统和编译器规定有关，可在编译器上查看。 ） sizeof(int)
    -基本型
    int 占4个字节
    -短整型
    short int | short
    -长整型
    long int | long
    -无符号型
    unsigned

    实型：
    实数、浮点数，C语言中实数只采用十进制。
    -十进制小数形式 必须有小数点
    0.0 25.0 5.669 0.25 
    -指数形式 加阶码 e | E
    aEn (a为十进制数，n为十进制整数)
    2.1E5 (2.1 * 10的5次幂)
    3.7E-2 (3.7 * 10的-2次幂)

    浮点数允许使用后缀：f | F
    如356f和356.是等价的


    实型在内存中的存放形式：
    4个字节，32位内存空间。按指数形式存储。
    3.14159在内存中存放形式如下
    符号  小数部分  指数部分
    +     .314159  1
    小数部分占的bit位数越多，精度越高
    指数部分占得位越多，数值范围越大

    实型分类
    单精度
    float
    双精度
    double
    长双精度
    long double

    作业：算各个类型数值范围、有效数值范围、字节数
    
  3.补充
    内存中的二进制存储
    一个字节 byte = 8 bit
    数值是已补码表示的
    正数的补码和原码相同
    负数的补码：将该数的绝对值的二进制，按位取反再加1
    
    例如：求-10的补码       8 bit
    10的原码：             00001010
    取反：                 11110101
    加1，得-10的补码:      11110110
    
```

```
    多个基本数据类型或构造类型。

    - 数组类型
    - 结构体类型
    - 共用体（联合）类型

  3.指针类型
    特殊的、具有重要作用的数据类型，其值用来表示某个变量在内存中的地址。
  
  4.空类型
    void
  
```
