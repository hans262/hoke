## 内存
```
  1.内存原理
    内存以二进制方式存储
    1kb = 1024 byte
    1个字节(byte) = 8位(bit)
    数值是已补码表示的
    正数的补码和原码相同
    负数的补码：将该数的绝对值的二进制，按位取反再加1
    
    例如：求-10的补码       8 bit
    10的原码：             00001010
    取反：                 11110101
    加1，得-10的补码:      11110110

  2.常量
    程序执行过程中，其值不发生改变
    #define 名称 值
```

## 进制
```
  1.二进制 ->
  0b00000010 = 2
  0b00000011 = 3

  2.八进制 -> 0开头, 0 ~ 7
  0o10 = 8
  0o11 = 9

  3.十六进制 -> 0x开头, 0 ~ 15, A-F(0~15)
  0x10 = 16
  0x20 = 32
  0xA0 = 160
```

## malloc
```
  void *malloc(size_t size)
  动态分配所需的内存空间，并返回一个指向它的指针

  (int *)malloc(sizeof(int) * 2);
  
  分配两个int的空间，然后把返回值强转为int形指针

```
