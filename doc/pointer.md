## 指针 pointer
```
  1.含义
  -指针是一种保存变量地址的变量
  -其值用来表示某个变量在内存中的地址
  -每一个变量都有一个内存位置
  -可使用连字号'&'访问变量的内存地址

  2.使用
  int * p;
  声明一个int类型的指针

  int num = 10;
  p = &num;
  获取num的内存地址，然后复制给指针变量p;

  cout << *p << endl;
  使用指针访问其值;
  
```

## 内存
```
  1.内存地址
  计算机的每一个内存单元（字节），都有一个固定的编号。
  通过这个这个编号（内存地址），就可以访问该地址所存储的值。

  2.空指针
  地址：00000000
  用NULL来表示

  为指针变量赋一个 NULL 值是一个良好的编程习惯
  可用if来判断是否是空指针

  3.指针运算
  + - ++ --
  内存地址的偏移
    
  ++p
  假设p指向地址1000，指针类型为int，每个指针占4个字节 
  运算结束后，p指向1004地址

```
