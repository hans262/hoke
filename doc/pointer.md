## 指针

```
  pointer
  特殊的、具有重要作用的数据类型，其值用来表示某个变量在内存中的地址。
  
  每一个变量都有一个内存位置
  可使用连字号（&）访问变量的内存地址

  含义：
  指针是一个变量，其值为另一个变量的内存地址，内存位置的直接地址。

  申明指针变量：
  int *p; -> 一个整型指针

  指针的类型：
  取值的时候，我要知道怎么取：4字节、8字节、16字节
  指针的偏移，指针移动时，如果是int类型指针，按照4字节移动

  获取指针：
  &num;  -> 获取变量num的内存地址

  使用指针访问其值：
  *p
  
```

```
  内存地址

  计算机的每一个内存单元（字节），都有一个固定的编号。
  通过这个这个编号（内存地址），就可以访问该地址所存储的值。

```

```
  空指针
  地址：0000000000000000

  为指针变量赋一个 NULL 值是一个良好的编程习惯
  一般用来做if判断
  
```

```
  指针运算 + - ++ --
  内存地址的偏移

  p++
  假设p指向地址1000，每个指针占4个字节，
  运算结束后，p指向1004地址。
  
```
