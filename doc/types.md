### 其他
```
  1.内存原理
    内存中的二进制存储
    一个字节 byte = 8 bit
    数值是已补码表示的
    正数的补码和原码相同
    负数的补码：将该数的绝对值的二进制，按位取反再加1
    
    例如：求-10的补码       8 bit
    10的原码：             00001010
    取反：                 11110101
    加1，得-10的补码:      11110110

  2.常量
    程序执行过程中，其值不发生改变
    #define 名称 值

  3.进制
    十进制 -> 0 ~ 9
      范围 0 ~ 65535 | -32768 ~ 32767
    八进制 -> 0开头，0 ~ 7
      015(十进制为13) 0101(十进制为65) 0177777(十进制为65635)
      范围 0 ~ 0177777
    十六进制 -> 0X或0x开头，0 ~ 9 | A ~ F | a ~ f
      0X2A(十进制为42) 0XA0(十进制为160) 0XFFFF(十进制为65535)
      范围 0X0 ~ 0XFFFF | 0x0 ~ 0xFFFF
```

### malloc
```
  void *malloc(size_t size)
  动态分配所需的内存空间，并返回一个指向它的指针

  (int *)malloc(sizeof(int) * 2);
  
  分配两个int的空间，然后把返回值强转为int形指针
```
